# Bikeshare Project

This project analyzes bikeshare data to explore how users interact with bike-sharing systems.
It allows filtering data by city, month, and day, and provides statistics on travel time,
popular stations, trip duration, and user demographics.

## Requirements
- Python 3.x
- pandas
- numpy

Install required libraries using:
pip install pandas numpy

## How to Use
1. Run the `bikeshare.py` script.
2. Select a city for analysis.
3. Choose a month and day to filter the data.
4. Review the statistics displayed in the terminal.

## Project Structure
- bikeshare.py: Main script that runs the analysis
- README.md: Project documentation
- .gitignore: Excludes CSV data files from version control
